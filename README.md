🍕 Pizza Sales Analysis
📌 Overview

This project focuses on analyzing pizza restaurant sales data to extract meaningful insights that support business decision-making.
The entire analysis was performed using Excel and Power Query for data cleaning, transformation, and exploration.

🎯 Objectives
- Analyze overall sales performance
- Identify top-selling pizza types
- Understand customer purchasing behavior
- Detect peak hours and high-demand periods
- Support business decisions with data insights
  
🛠️ Tools Used
- Excel → Data cleaning and analysis
- Power Query → Data transformation and preparation

🔄 Project Workflow
1. Data Cleaning
. Removed duplicate records
. Handled missing values
. Standardized data formats (dates, prices)
2. Data Transformation
. Extracted time-based features (day, month)
. Categorized pizza types and sizes
. Created calculated columns such as:
. Total Sales
. Number of Orders
3. Data Analysis
. Analyzed sales by:
. Pizza type
. Size
. Time (daily / monthly trends)
Identified:
. Best-selling products
. Low-performing items

📊 Key Insights
- Sales peak during evening hours
- A few pizza categories generate the majority of revenue
- Some items consistently underperform
- Customer ordering behavior varies by time of day

💡 Recommendations
- Focus marketing efforts on top-selling pizzas
- Introduce promotions during low-demand periods
- Optimize the menu by improving or removing weak items
- Prepare for peak hours to maximize efficiency


🚀 Project Value

This project demonstrates the ability to:

- Clean and analyze data using Excel
- Transform data using Power Query
 Extract actionable insights from raw data
- Support business decisions using data
